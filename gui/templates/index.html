Python (OpenAI SDK) 
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:8080/v1",
    api_key="free"  # orice șir de caractere
)

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Salut!"}],
    stream=True,
)

for chunk in response:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")

Listă endpoint-uri

GET /v1/models Obține lista modelelor disponibile
POST /v1/chat/completions Completare conversație (suportă flux)
Parametri cerere
modelstringobligatoriuNume model, ex. "gpt-4o-mini"
messagesarrayobligatoriuListă mesaje [{role, content}]
streambooleanopționalIeșire în flux, implicit false
temperaturefloatopționalParametru temperatură, 0-2, implicit 0.7
max_tokensintegeropționalNumăr maxim token-uri ieșire, implicit 4096
providerstringopționalSpecifică furnizorul (câmp extins)
GET /api/providers Obține starea tuturor furnizorilor
POST /api/providers/{name}/toggle Activează/dezactivează furnizor specific
GET /api/stats Obține statistici generale
